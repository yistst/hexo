Termux挂网易云脚本

安装

点这进F-Deoid下载Termux
安装好后打开Termux软件
执行下面指令更新源，安装python，安装git

‘’’bash
apt update -y
apt install -y python openssh git
‘’’

拉取库到本地

‘’’bash
git clone https://github.com/chen310/NeteaseCloudMusicTasks.git
‘’’
切换到项目目录

‘’’bash
cd NeteaseCloudMusicTasks
‘’’
然后安装依赖

‘’’bash
pip install -r requirements.txt
‘’’

修改配置文件

首先将 config.example.json 文件复制为 config.json 文件

‘’’bash
cp config.example.json config.json
‘’’

然后对配置文件 config.json 进行修改。
运行
‘’’bash
python3 index.py
‘’’
